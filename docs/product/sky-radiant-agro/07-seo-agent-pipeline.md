# SEO Agent Pipeline

## Objective

Position Sky Radiant Agro as the trusted data intelligence platform for agri procurement and trade in North India.

## Weekly Content Cadence

- 3 commodity intelligence reports
- 2 procurement explainers
- 1 buyer/farmer success story
- 1 market outlook note

## Workflow

1. Topic Discovery
   - Pull top price-moving commodities.
   - Pull top weather-risk regions.
   - Pull seasonal demand trends.

2. Keyword Mapping
   - Primary keyword
   - Secondary keyword cluster
   - Search intent (informational/commercial)

3. Content Brief Generation
   - Target persona
   - Key questions answered
   - Data visuals required
   - CTA

4. Draft Generation
   - SEO title
   - Meta description
   - H1/H2/H3 hierarchy
   - Internal links

5. Publish and Distribute
   - Website blog
   - LinkedIn
   - YouTube shorts script from same content
   - WhatsApp channel summary

6. Measure and Iterate
   - impressions
   - clicks
   - CTR
   - rank changes
   - lead conversions

## High-Value Topic Buckets

- "Best time to procure [commodity] in North India"
- "Mandi price forecast for [commodity]"
- "Cold storage vs dry storage guide for [commodity]"
- "How weather impacts [commodity] pricing"
- "Export opportunity for [commodity] from India"

## KPI Targets (First 90 Days)

- 100 indexed pages
- 30 keywords in top-20
- 10 high-intent keywords in top-10
- 3x organic inbound leads
