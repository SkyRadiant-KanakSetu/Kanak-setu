# PRD Summary

## Problem
Donors need trusted, transparent donation rails for gold-backed contributions to verified institutions.

## Product Outcome
Enable donor-to-institution digital gold donations with traceable payment events, auditable operations, and blockchain-verifiable proof.

## Primary Users
- Donor
- Institution admin
- Platform admin and operators

## MVP Scope
- Donor donation creation and history
- Institution onboarding/review workflow
- Payment webhook processing with idempotency and signature checks
- Gold allocation and institution ledger updates
- Merkle batching, chain anchoring, and proof retrieval
- Certificate issuance and verification endpoint
- Admin reconciliation and operational controls
