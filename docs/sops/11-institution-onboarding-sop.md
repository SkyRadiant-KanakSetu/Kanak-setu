# Institution Onboarding SOP

## Steps for Institution Admin
1. Register institution admin account.
2. Complete institution profile and upload required documents.
3. Submit onboarding profile for review.
4. Track status updates in institution portal.

## Status Lifecycle
- `DRAFT` -> `SUBMITTED` -> `UNDER_REVIEW` -> `ACTIVE`
- Alternate outcomes: `REJECTED` or `SUSPENDED`

## Required Data
- Public institution name
- Legal entity details
- Bank account details
- Compliance/KYC documents

## Failure Handling
- If rejected, update requested fields and resubmit.
- If suspended, coordinate with compliance team before reactivation.
